#  爬山演算法  
```
種局部擇優的方法，採用啟發式方法  
一般存在以下問題：  
局部最大  
高地：也稱為平頂，搜尋一旦到達高地，就無法確定搜尋最佳方向，會產生隨機走動，使得搜尋效率降低。  
山脊：搜尋可能會在山脊的兩面來回震盪，前進步伐很小。  
解決方法：隨機重新啟動爬山演算法
資料來源:https://zh.m.wikipedia.org/zh-tw/%E7%88%AC%E5%B1%B1%E7%AE%97%E6%B3%95
```
